This Website!

My current personal project is the design and implementation of this website as a part of the class I TA for, computing in the modern world (CSCE 190). It is coded in html, css, and javascript, and designed in Figma. Because this is a freshman level course, I wanted the tech stack to have a low barrier of entry so students could see what they could develop with the code. This website is hosted on Github in a public repository linked here. In this project I am learning how to develop more complex and interactive components, and I am also learning to create responsive web design.
